HTML Links - Different Colors

HTML Link Colors

Por padrão, um link aparece nos navegadores da seguinte forma:

Um link não visitado está sublinhado e azul

Um link visitado é sublinhado e roxo

Um link ativo está sublinhado e vermelho

Agora mudando os padrões com css

Este link não visitado ficará verde e sem sublinhado (a:link)

Este link visitado será azul sem sublinhado (a:visited)

Este link ativo será amarelo e sublinhado (a:active)

Todos os links ao passar o mouse, ficarão vermelho e sublinhado (a:hover)
